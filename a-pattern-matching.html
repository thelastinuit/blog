So, I found the following function:

def set_class_labels(tuples) do
  tuples
  |> Map.keys()
  |> Enum.map(fn val ->
    case val do
      99 -> "Tournaments"
      1000 -> "Open"
      val -> val
    end
  end)
end

The function is using Map.keys(param) which needs a map, no tuple. Also, the function is using a case withing Enum.map. This an opportunity for pattern matching super powers.

Let us start by rename the parameter:

def set_class_labels(map) do
  map
  |> Map.keys()
  |> Enum.map(fn val ->
    case val do
      99 -> "Tournaments"
      1000 -> "Open"
      val -> val
    end
  end)
end

Following, we can get rid of the case since it's not necessary:

def set_class_labels(map) do
  map
  |> Map.keys()
  |> Enum.map(fn
    99 -> "Tournaments"
    1000 -> "Open"
    val -> val
  end)
end
